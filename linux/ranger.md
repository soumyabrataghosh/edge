# Ranger

## Keybindings

| Key   | Command                    |
| ---   | -------                    |
| Space | Mark file                  |
| v     | Mark all files             |
| dd    | Cut                        |
| yy    | Copy                       |
| ya    | Add to current `copy`      |
| yr    | Remove from current `copy` |
| uv    | Unmark                     |
| cw    | Rename                     |
| I     | Rename from the beginning  |
| A     | Rename from the end        |
| mX    | Create a bookmark          |
| `X    | Go to bookmark             |
| zf    | Filter the file list       |
| o     | Change sort order          |
|       |                            |
| zh    | View hidden files          |
| /     | Search                     |
| cd    | Change directory           |
